## Quick Start
```bash
# 1. Clone
git clone <remote-url>
cd patina-design

# 2. Install
pip install -r requirements.txt  # or: npm install

# 3. Configure
cp .env.example .env
# Edit .env with your values

# 4. Run
python main.py  # or: npm start
```
